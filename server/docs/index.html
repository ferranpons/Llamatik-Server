Llamatik API

Lightweight llama.cpp HTTP server exposing the same surface as the Llamatik LlamaBridge API.

  • JSON endpoints for embeddings and generation
  • SSE streaming endpoints for token/delta streaming
More information: https://helloreverb.com
Contact Info: hello@helloreverb.com
